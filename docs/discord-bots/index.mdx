## Key Differences

<AccordionGroup>
  <Accordion title="Email Bot - Email to Discord">
    **What it does**: Creates unique email addresses for Discord channels. Any emails sent to these addresses automatically appear in your channel.

    **Best for**:
    - Receiving notifications from services (monitoring, billing, GitHub, etc.)
    - Website contact form submissions
    - Newsletter distribution to teams
    - Server alerts and CI/CD notifications
    - Gmail forwarding with filters

    **How it works**: `/connect` command generates a unique email address → Send emails to that address → They appear in Discord
  </Accordion>

  <Accordion title="Telegram Bot - Telegram to Discord">
    **What it does**: Bridges Telegram and Discord, forwarding Telegram conversations to Discord threads where teams can reply.

    **Best for**:
    - Customer support via Telegram
    - Managing Telegram communities from Discord
    - Multi-platform support operations
    - Team collaboration across platforms
    - Monitoring Telegram groups/channels

    **How it works**: Connect your Telegram bot token → Telegram messages create Discord threads → Reply from Discord, messages go to Telegram
  </Accordion>
</AccordionGroup>

## Quick Comparison

| Feature | Email Bot | Telegram Bot |
|---------|-----------|--------------|
| **Primary Function** | Email → Discord | Telegram ↔ Discord |
| **Communication** | One-way or two-way email forwarding | Two-way real-time messaging |
| **Use Case** | Notifications, forms, alerts | Customer support, conversations |
| **Setup Complexity** | Simple (one command) | Moderate (requires Telegram bot token) |
| **Organization** | All emails in channel | Each conversation in separate thread or channel |
| **Media Support** | Attachments, images | Photos, videos, voice, stickers, documents |
