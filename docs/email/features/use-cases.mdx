Email Bot bridges email and Discord, enabling countless integration possibilities. Here are the most common and effective use cases to inspire your setup.

## Customer Support

Forward customer inquiries from your website contact form directly to a Discord support channel.

### Setup

<Steps>
  <Step title="Create Support Channel">
    Set up `#customer-support` or `#inquiries` channel in Discord
  </Step>
  <Step title="Connect Bot">
    Run `/connect` to get unique email address
  </Step>
  <Step title="Configure Contact Form">
    Update website contact form to send submissions to bot's email
  </Step>
  <Step title="Enable Preview">
    Turn on "Email Preview" setting for immediate context
  </Step>
  <Step title="Set Up Threads">
    Use Discord threads to organize responses per inquiry
  </Step>
</Steps>

### Benefits

- **Faster Response**: Team sees inquiries immediately in Discord
- **Collaborative**: Multiple team members can discuss and respond
- **Organized**: Each inquiry becomes its own thread
- **Transparent**: Everyone sees customer issues and resolutions

### Recommended Settings

- **Display Mode**: Subject + Preview
- **Web Version**: Enabled (for form-rich submissions)
- **Personalization**: "Customer Support" with support icon avatar
- **Threads**: Enabled for per-inquiry discussions

## Gmail Forwarding

Automatically forward specific Gmail messages to Discord channels.

### Use Cases

- Forward emails from specific senders
- Route emails with certain keywords
- Create department-specific channels
- Archive important emails in Discord

### Setup

<Steps>
  <Step title="Open Gmail Settings">
    Gmail → Settings (gear icon) → See all settings
  </Step>
  <Step title="Go to Forwarding">
    Click "Forwarding and POP/IMAP" tab
  </Step>
  <Step title="Add Forwarding Address">
    Click "Add a forwarding address"
    Enter bot's email from `/connect`
  </Step>
  <Step title="Verify in Discord">
    Gmail sends verification email to Discord channel
    Click verification link in Discord message
  </Step>
  <Step title="Enable Forwarding">
    Back in Gmail, select "Forward a copy of incoming mail to..."
    Choose bot's email address
  </Step>
  <Step title="Create Filters (Optional)">
    Set up Gmail filters to forward only specific emails
  </Step>
</Steps>

### Filter Examples

**Forward from specific sender**:
```
From: boss@company.com
Action: Forward to [bot email], Skip inbox
```

**Forward by keyword**:
```
Subject: [URGENT]
Action: Forward to [bot email]
```

**Forward by label**:
```
Has label: Important
Action: Forward to [bot email]
```

<Tip>
Learn more about Gmail filters: [Gmail filter guide](https://support.google.com/mail/answer/6579)
</Tip>

## GitHub Notifications

Receive repository notifications, pull request updates, and issue comments in Discord.

### Setup

<Steps>
  <Step title="Create GitHub Channel">
    Set up `#github-notifications` in Discord
  </Step>
  <Step title="Connect Bot">
    Get email address with `/connect`
  </Step>
  <Step title="Configure GitHub Notifications">
    1. Go to GitHub → Settings → Notifications
    2. Set email to bot's address
    3. Choose notification preferences
  </Step>
  <Step title="Repository-Specific">
    For specific repos: Repository → Settings → Webhooks
    Configure email notifications
  </Step>
</Steps>

### What You'll Receive

- Pull request creation and updates
- Issue comments and mentions
- Release notifications
- Security alerts
- Dependabot updates
- Code review requests

### Recommended Settings

- **Display Mode**: Subject Only (high volume)
- **Web Version**: Enabled (for code diff viewing)
- **Personalization**: "GitHub" with GitHub logo
- **Organization**: Use threads for each PR/issue

## Billing & Payment Notifications

Centralize billing alerts, payment confirmations, and subscription renewals.

### Common Services

- Stripe payment notifications
- Domain registration renewals
- Cloud hosting bills (AWS, DigitalOcean, Heroku)
- SaaS subscription renewals
- Invoice receipts

### Setup

<Steps>
  <Step title="Create Billing Channel">
    Set up `#billing` or `#payments` in Discord
  </Step>
  <Step title="Get Email Address">
    Use `/connect` command
  </Step>
  <Step title="Update Service Emails">
    For each service:
    1. Go to account settings
    2. Update notification email
    3. Enter bot's email address
  </Step>
  <Step title="Enable Web Version">
    Turn on web version for invoice viewing
  </Step>
</Steps>

### Benefits

- Never miss renewal notifications
- Team visibility into expenses
- Quick access to invoices
- Centralized billing records

### Recommended Settings

- **Display Mode**: Subject + Preview
- **Web Version**: Enabled (for invoice viewing)
- **Personalization**: "Billing" with dollar sign icon
- **Archive**: Pin important payment confirmations

## Form Submissions

Receive website form submissions directly in Discord.

### Compatible Forms

- Contact forms
- Job applications
- Event registrations
- Survey responses
- Newsletter signups
- Support tickets

### Setup

<Steps>
  <Step title="Choose Form Service">
    Examples: Typeform, Google Forms, Wufoo, JotForm
  </Step>
  <Step title="Create Forms Channel">
    Set up dedicated channel like `#form-submissions`
  </Step>
  <Step title="Connect Bot">
    Get email with `/connect`
  </Step>
  <Step title="Configure Form">
    In form settings, set notification email to bot's address
  </Step>
  <Step title="Test Submission">
    Submit test form to verify delivery
  </Step>
</Steps>

### Example (Google Forms)

<Steps>
  <Step title="Open Form Settings">
    In Google Forms editor, click Settings gear
  </Step>
  <Step title="Go to Responses">
    Click "Responses" tab
  </Step>
  <Step title="Get Email Notifications">
    Enable "Get email notifications for new responses"
  </Step>
  <Step title="Set Email">
    Cannot set custom email directly - use Gmail forwarding instead
  </Step>
</Steps>

- GitHub Actions

- Vercel

### Setup

## E-commerce Order Notifications

Forward order confirmations, shipping updates, and customer notifications.

### Platforms

- Shopify
- WooCommerce
- BigCommerce
- Etsy
- eBay
- Amazon Seller Central

### Setup

<Steps>
  <Step title="Create Orders Channel">
    Set up `#orders` in Discord
  </Step>

- **Display Mode**: Subject + Preview
- **Web Version**: Enabled (for order details)
- **Personalization**: "Orders" with shopping cart icon
- **Threads**: Group by order number
