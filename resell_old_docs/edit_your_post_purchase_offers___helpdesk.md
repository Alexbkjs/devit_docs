# Edit your post-purchase offers

## Edit offers

ReSell application is a flexible app and provides customization features and allows you to edit your post-purchase offer. Near the selected product click **Edit offer** and you will be moved to the page where you can customize this offer.

![](https://storage.crisp.chat/users/helpdesk/website/-/3/2/7/1/3271b421d88bd400/edit-offer_1sohf93.png)

## ✔️ Edit existing elements

Click on the element that you want to edit and in the opened right-side bar you can change its title, content, text size and alignment, disable / enable border and background.

![](https://storage.crisp.chat/users/helpdesk/website/-/3/2/7/1/3271b421d88bd400/edit-element_1uvlm8w.png)

The **Product** section provides a more extensive list of settings. Here you can change the product, add custom description, hide / display product variants and tax lines, apply discount, and change main product image.

## ✔️ Add new elements

To add a new elements / widgets to your offer, click **Add element** and select widgets that you need.

![](https://storage.crisp.chat/users/helpdesk/website/-/3/2/7/1/3271b421d88bd400/add-new-widget_1jprump.png)
